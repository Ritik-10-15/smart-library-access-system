# Smart Library Access System 
A c++ console-based project that simulates a start library system.The system manages different types of library users and library items using object-oriented programming principles.

## Features
- supports multiple library items(books,laptops,study rooms)
- supports different user roles(students, faculty, guest)
- Enforces borrowing rules using polymorphism
- Tracks due dates and late fees

## OOP Concepts Used
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Enum classes

## Project Structure
- include/:Header files
- src/:Source files
- README.md: Project documentation

## How to Compilen and run 
- From the src folder: g++ -I../include *.cpp -o program.exe
- ./program.exe
